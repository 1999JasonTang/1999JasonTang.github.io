Project Showcase

Rainy

PyTorch Image Classification Algorithm

One of my favorite things in the world is rain. So as my first project in computer vision, I used the Resnet 18 model (link to paper) trained on imagenet for feature extraction and made a custom fully connected top layer to classify whether or not an image contains rain. I trained using a dataset of ~900 images scraped off of the raining subreddit in order to achieve an accuracy of 95.8%.

How Much Do I Need

Java Android Application

My first crack at an Android Application, it keeps track of current grades and calculates your current mark based off the syllabus and current known grades. It's main purpose is to calculate how much is needed on a final exam to end the course with a certain grade, something that I do manually every time. I initially wrote it in python but then ported it over to mobile for convenience.

Field Team Management System

React Web Application

As part of the Technology Leadership Initiative, I am one of the 24 students taking courses modified for an industry integrated learning experience. We developed this project as a Human Resource management tool for Nor-weld Ltd, an Ontario based Welding company. It's main functions are to manage client work orders, assign jobs, and submit timesheets.
